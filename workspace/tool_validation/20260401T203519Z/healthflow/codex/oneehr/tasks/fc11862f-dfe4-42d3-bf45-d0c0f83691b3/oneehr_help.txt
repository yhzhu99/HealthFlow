usage: oneehr [-h] {preprocess,train,test,analyze,plot,convert} ...

OneEHR CLI

positional arguments:
  {preprocess,train,test,analyze,plot,convert}
    preprocess          Bin features, generate labels, split patients
    train               Train ML/DL models
    test                Run all systems on test set
    analyze             SHAP, fairness, cross-system comparison
    plot                Render publication-quality figures
    convert             Convert a raw dataset into OneEHR three-table format

options:
  -h, --help            show this help message and exit
